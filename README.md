# mtg_card_evaluator
cs229 class project

Predicts the value of Magic; the Gathering cards based on their text.

## Instructions

unix: 
<ol>
<li>clone the repository </li>
<li>bash run.sh </li>
</ol>

windows: 
<ol>
<li>clone the repository</li>
<li>unzip AllSets.json.gz => AllSets.json </li>
<li>run python features/generate.py </li>
</ol>
